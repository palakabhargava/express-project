# Express Project

A REST API backend built using Node.js, Express.js, MongoDB, and JWT Authentication.

## Features

- User Registration
- User Login
- JWT Authentication
- Protected Routes
- Contact Management CRUD Operations
- Error Handling Middleware
- MongoDB Integration

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication

## Project Structure

├── config
├── controller
├── middleware
├── models
├── routes
├── server.js

## API Endpoints

### User Routes

POST /api/users/register

POST /api/users/login

GET /api/users/current

### Contact Routes

GET /api/contacts

POST /api/contacts

PUT /api/contacts/:id

DELETE /api/contacts/:id


## Installation

```bash
git clone <repository-url>
cd express-project
npm install
npm start
```
